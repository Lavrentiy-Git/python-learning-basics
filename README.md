# Python Learning Basics

Репозиторий для систематической практики основ Python (март 2026 — ...).

*Цель*:  
От простых функций и модулей → к реальным проектам автоматизации и AI-инструментам для фриланса.

Что внутри
- День 1: Функции, return, тесты в `if __name__ == "__main__":`  
  Файл: `day1_functions.py`
- День 2: Сортировка, работа со списками и строками
  Файл: `day2_functions.py`
  Файл: `christmas_tree_animation_day2.py`
- Будущие дни: работа с файлами, исключениями, регулярными выражениями, OOP, requests, pandas и первые AI-скрипты
